# Jency's Git Exercise
Basic Commands Explained
git status ->shows which files are changed, staged, or ready to commit
git add <file>  → Marks a file to be included in your next commit
git commit -m 'message'  → Saves your changes locally with a message
git log  → Shows the history of commits
git log --oneline  → Shows history in short, one line per commit
git branch  → Lists all branches
git switch <branch>  → Moves you to another branch
git switch -c <branch>  → Creates a new branch and switches to it
git push  → Uploads your commits to GitHub
git pull  → Downloads updates from GitHub
git clone <repo>  → Downloads a repo to your computer
